# EKG & Aktivitäts-Analyse App

Diese interaktive Streamlit-App analysiert Fitnessdaten (EKG und Leistung). Sie berechnet Leistungsstatistiken, teilt die Aktivität basierend auf einer dynamisch wählbaren maximalen Herzfrequenz in 5 Belastungszonen ein und visualisiert die Daten in einem interaktiven Plotly-Diagramm.
